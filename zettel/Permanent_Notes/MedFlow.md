---
Nome do Projeto: MedFlow
Tecnologias: Java
Status: Ativo
Objetivo: Um sistema que contenha dados do paciente, que seja de fácil acesso e seguro
---
## Resumo

Um sistema que contenha:
* Diagnóstico
* Histórico clínico 
* Evolução
* ID único do paciente 
* Cor da pulseira(Classificação de risco)
* Tipo sanguíneo(se possível)
* Alergias a medicamentos(se possível)

## Funcionalidades

 * Qrcode de fácil acesso com dados importantes do paciente
 * Cadastro do paciente
