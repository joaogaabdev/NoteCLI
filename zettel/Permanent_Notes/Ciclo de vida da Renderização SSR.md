#Frontend 
1. **Requisição HTTP**
    
    - O navegador faz uma requisição para uma rota (`/`, `/produtos`, etc.).
        
2. **Renderização no Servidor**
    
    - O servidor obtém os dados da API e renderiza o HTML completo.
        
3. **Envio do HTML**
    
    - O HTML é enviado ao cliente (renderização instantânea).
        
4. **Hidratação (Hydration)**
    
    - O bundle JS é carregado e conecta eventos e estados.
        
5. **Interatividade**
    
    - A aplicação passa a se comportar como uma SPA.
