# 🔢 Variáveis, Tipos de Dados e Operações

## Tipos Primitivos

```c
int idade = 20;
float altura = 1.75;
char letra = 'A';
```

## Entrada e Saída

```c
int x;
scanf("%d", &x);
printf("Valor: %d", x);
```

## Operadores

- Aritméticos: `+`, `-`, `*`, `/`, `%`
- Relacionais: `==`, `!=`, `<`, `>`
- Lógicos: `&&`, `||`, `!`
