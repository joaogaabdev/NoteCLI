
## API
Application Programming Interface - Interface de Programação da Aplicação

São conjuntos de rotinas documentadas e disponibilizadas por uma aplicação para que outras aplicações possam consumir suas funcionalidades.

Quando uma aplicação web disponibiliza um conjunto de rotinas e padrões através de serviços web podemos chamar esse conjunto de API.

Utiliza dos principais Métodos HTTP   

GET - Solicita um recurso
POST - Solicita a criação de um recurso
PUT - Solicita a atualização de um recurso
DELETE - Solicita a exclusão de um recurso
