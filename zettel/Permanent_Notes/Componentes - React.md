## Conceitos Fundamentais

## JSX (JavaScript XML)

É uma extensão de sintaxe do JavaScript que permite escrever código parecido com HTML dentro do JS.

`function Hello() {   return <h1>Olá, React!</h1>; }`

O JSX é convertido em chamadas `React.createElement()` na build.  
Ele une lógica e UI no mesmo arquivo, ao contrário do modelo antigo de separar HTML, CSS e JS rigidamente.
