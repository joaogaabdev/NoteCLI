#Frontend

React é uma **biblioteca JavaScript para construção de interfaces de usuário (UI)**, criada pelo **Facebook (hoje Meta)** em 2013.  
O foco é **criar componentes reutilizáveis**, que representam partes independentes da interface, e que se atualizam de forma eficiente quando os dados mudam.

- ⚙️ **Tipo:** Biblioteca (não framework completo)
    
- 💡 **Principal paradigma:** _Programação declarativa_ e _componentização_
    
- 🧱 **Base:** JavaScript + JSX
    
- 🧭 **Licença:** MIT (livre pra uso comercial e open source)
