## Loops em GO

Em Go não temos While, para loops, temos basicamente o for.
Podemos iterar com o uso de range.
Usar break e continue aninhados.
``` go
	for i:= 0; i<5; i++{
		fmt.Prinln(i)}
```

## *For* como *While* 
* Em Go não temos o *while*, mas podemos utilizar o for como um.

*Break* e *Continue*
* Podemos utilizar o *Break* para parar o código e o *Continue* para "pular"
